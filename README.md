# 🌱 AgroBot - AI Powered Plant Disease Detection Assistant

AgroBot is a multilingual AI-powered web application that helps farmers and agricultural enthusiasts identify plant diseases using image-based diagnosis and an intelligent chatbot assistant.

## Installation

### Clone Repository

```bash
git clone https://github.com/Subhrajit567/AgroBot.git
cd AgroBot
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
